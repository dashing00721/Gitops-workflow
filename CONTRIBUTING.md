# Contributing Guide

## How to work on this project
1. Pick a task from Trello
2. Create a branch
3. Write your code
4. Push to GitHub
5. Open a Pull Request
6. Wait for green CI
7. Merge it

## Branch naming rule
feature/TRELLO-001-what-you-are-doing

## Commit message rule
[TRELLO-001] What you did

## If CI goes red
- Click the red X in Actions tab
- Read the error
- Fix it locally
- Push again
